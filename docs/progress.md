## 当前完成度

### 2026-07-17 — 管理员模式、任务清理与执行页复检

- 历史任务的单次抽屉与批次详情 GPU 面板补齐远端 `nvidia-smi` 运行指标：风扇转速、实时/上限功耗、性能状态和 PCIe Bus-ID；接口采用可选新增字段，保持旧客户端兼容。
- 顶部新增普通/管理员模式切换。管理员会话有效期为 5 分钟，显示倒计时；使用 HttpOnly Cookie 支持刷新恢复，手动退出或超时后保持普通模式。
- 普通模式保留新增/编辑服务器、部署公钥、上传脚本密码确认、提交/取消任务等日常操作；管理员模式负责删除与系统设置类高风险操作。
- 审计日志在管理员模式下直接加载；普通模式点击侧栏入口显示权限提示。
- 历史任务支持删除单任务及整批任务：删除本机 artifacts、日志与任务历史，保留远端目录和审计日志。
- 执行任务页进入时只复检当前在线服务器的 SSH 可达性，避免为离线服务器等待超时。
- 管理员模式侧栏使用金色权限主题；普通模式菜单使用蓝色选中态；工作区与卡片采用低饱和暖金层次。

HPCDeploy 已形成完整闭环，端到端链路全部打通：

```
服务器管理（含标签筛选/健康探测/一键部署公钥）
→ 脚本/模板选择
→ 多服务器任务执行
→ 实时日志 + 实时监控
→ 结果文件回收与下载
→ 失败诊断
→ 审计日志
→ 系统设置（运行数据/本机清理）
```
