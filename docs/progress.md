## 当前状态

HPCDeploy 已形成完整可用 MVP，端到端链路全部打通：

```text
服务器管理 → 脚本知识库 → 选择服务器（支持多选批量）
→ 选择脚本（支持模板预设） → SSH 上传 → 远程执行
→ 实时日志 → 资源快照 → 任务历史 → 结果文件回收
→ 取消任务 → 删除任务 → 仪表盘总览 → 结果文件归档目录查看
```

核心增强：
- **探测全部**：并发 ThreadPoolExecutor + offline 跳过（默认）+ 3s timeout，默认总耗时 < 1s
- **脚本模板**：9 个任务模板（6 stress / 2 mpi / 1 apptainer）自动填充参数
- **批量执行**：多选服务器 → 独立 task_id → 独立并发线程 → 部分成功
- **安全白名单**：params 禁止 command/raw_args/shell/raw_command，stress 参数值白名单校验
