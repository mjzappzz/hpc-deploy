## 项目状态

所有 30 个阶段已全部完成；当前 v1.02 维护版已纳入服务器首次探测、公钥部署准入、任务重启恢复、GPU 软件部署和历史任务筛选体验的维护修复。后续不再新增阶段，以修复缺陷和优化体验为主。

---

## 明确当前不做

- Apptainer run/exec 执行
- 强制全站登录
- 复杂 RBAC / 多用户管理
- 调度器集群集成（Slurm 等）
- 可配置远端任务根目录

---

## 禁止事项

1. 禁止跳阶段一次性做完全部功能
2. 禁止前端直接执行 SSH
3. 禁止后端执行前端传入的任意命令
4. 禁止绕过脚本白名单
5. 禁止把 SSH 私钥返回给前端
6. 禁止没有日志的后台任务
7. 禁止任务没有状态
8. 禁止同一服务器无控制地并发执行
9. 禁止 apptainer run/exec
10. 禁止前端传 raw command / raw_args / remote_path / remote_work_dir
